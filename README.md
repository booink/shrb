# Shrb

Rubyで書かれたシェルです。
Bashの記法を出来るだけ再現するのが目標です。

Shell by Ruby.
The goal is to reproduce the notation of Bash as much as possible.

## インストール / Installation

    $ gem install shrb

## 使い方 / Usage

```sh
shrb
```
